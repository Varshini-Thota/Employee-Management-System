# Employee Management System

## Overview

The Employee Management System is a C programming project designed to manage employee records through a menu-driven interface. It allows users to add, view, search, update, and delete employee information efficiently.

## Features

* Add Employee Records
* View Employee Details
* Search Employee by ID
* Update Employee Salary
* Delete Employee Records
* Menu-Driven Interface
* Data Management Using Structures

## Technologies Used

* C Programming Language
* OnlineGDB

## How It Works

1. Users can add employee details including ID, name, and salary.
2. Employee records can be displayed and searched using Employee ID.
3. Existing employee salaries can be updated.
4. Employee records can be deleted when no longer required.
5. The system continuously runs until the user chooses to exit.
